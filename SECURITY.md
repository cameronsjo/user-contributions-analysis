# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities via [GitHub Security Advisories](https://github.com/cameronsjo/user-contributions-analysis/security/advisories/new).

**Do not** open a public issue for security vulnerabilities.

### Response Timeline

- **Acknowledgment:** within 48 hours
- **Assessment:** within 7 days
- **Fix/Disclosure:** within 30 days

## Disclosure Policy

We follow coordinated disclosure. We will credit reporters in the advisory unless they prefer to remain anonymous.
